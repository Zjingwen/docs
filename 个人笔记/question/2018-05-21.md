### Q：编程题
```javascript

    setTimeout(()=>resolve({name: 'y'}),1000);
});

const b = () => new Promise((resolve,reject)=>{
    setTimeout(()=>resolve({age: 10}),100);
});

function run(){
    a();
    b();
}
```
1. run()的执行时间大概是多少？
2. 修改run函数，将运行时间变成两个promise执行时间的总和
